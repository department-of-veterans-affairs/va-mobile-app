## Usage

## Accessibility considerations
* Refer to the [VA Design System for accessibility considerations](https://design.va.gov/components/pagination#accessibility-considerations)
* All interactive tappable elements should be coded as links and not as buttons.
* When additional context might be needed, we should apply a11yLabels to the links.
